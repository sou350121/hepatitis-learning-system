# Axis J: Ocular-Brain Connection (視網膜-大腦連接)

## 1. 核心邏輯 (Core Logic)
視網膜 (Retina) 在胚胎發育上是中樞神經系統 (CNS) 的延伸。大腦的病理變化（如 Aβ 沉積、血管退化、神經炎症）會同步反映在視網膜上。視網膜提供了一個「非侵入性、高分辨率」的窗口，用於失智症的超早期篩查。

## 3. 關鍵機制 (Key Mechanisms)

### A. 視網膜神經纖維層 (RNFL) 變薄
- **機制**：神經節細胞 (RGC) 的死亡早於大腦皮層的廣泛損失。通過光學相干斷層掃描 (OCT) 檢測 RNFL 厚度，可以預測未來 5-10 年的認知衰退。
- **2025 線索**：研究發現 RNFL 的特定象限（如顳側）變薄與海馬體萎縮高度相關。

### B. 視網膜微血管密度 (OCT-A)
- **機制**：大腦的 Axis H (血管界面) 退化會同步導致視網膜淺層毛細血管叢密度降低。
- **2024 情報**：在臨床症狀出現前，視網膜無血管區 (FAZ) 的擴大是 Aβ 陽性的強效預測因子。

---

## 4. 跨軸線交互 (Cross-Axis Interaction)
- **Axis J x Axis H**：視網膜血管是觀察全腦微循環的最佳樣本，可用於監測血管預修復治療的療效。
- **Axis J x Axis D**：利用 AI 深度學習分析視網膜底片，實現低成本的社區級大規模篩查。
