# Dementia Research Terminology (专业术语对照表)

> 为了保持学术研究的严谨性，本手册中的专业术语均保留英文原名，并在此提供详细的中文解析与逻辑背景。

| 术语 (Term) | 中文解析 (Explanation) | 所属维 (Axis) | 核心逻辑 (Core Logic) |
| :--- | :--- | :--- | :--- |
| **Aβ (Amyloid-beta)** | 类淀粉样蛋白 | Axis A | 在神经元外沉积形成“斑块 (Plaques)”，是 AD 的主要病理特征之一。 |
| **Tau Protein** | Tau 蛋白 | Axis A | 在神经元内过度磷酸化并形成“缠结 (Tangles)”，直接导致神经元死亡。 |
| **Proteostasis** | 蛋白质稳态 | Axis A | 维持蛋白质合成、折叠、运输与降解平衡的机制，其崩溃是退行性疾病的源头。 |
| **Primary Nucleation** | 初级成核 | Axis A | 单体蛋白自发结合形成第一个“种子 (Seed)”的过程，是聚合动力学速控步骤。 |
| **BBB (Blood-Brain Barrier)** | 血脑屏障 | Axis E/B/C | 由内皮细胞组成的物理与生化屏障，保护大脑免受血液中污染与病原体侵害。 |
| **Glymphatic System** | 类淋巴系统 | Axis A/C | 大脑的废物清除系统，主要在睡眠中运作，负责排出 Aβ 等代谢废物。 |
| **Microglia** | 微胶质细胞 | Axis B | 大脑的驻留免疫细胞，负责监视环境、吞噬斑块与引发炎症。 |
| **DAM (Disease-Associated Microglia)** | 疾病相关微胶质细胞 | Axis B | 一种特定的免疫状态，在病理环境下被激活，具有保护与毒性的双重性质。 |
| **Neuro-immune Niche** | 神经免疫生态位 | Axis B | 大脑内特定空间区域（如斑块周围），聚集了免疫细胞、神经元与病理蛋白的交互中心。 |
| **Gut-Brain Axis** | 肠脑轴 | Axis C | 肠道微生物及其代谢物透过迷走神经或循环系统与大脑进行双向通讯。 |
| **Irisin (鸢尾素)** | 肌肉激素 | Axis C | 运动时由肌肉分泌的一种激素，能穿透 BBB 并诱导大脑 BDNF 分泌。 |
| **BDNF** | 脑源性神经营养因子 | Axis C/A | 促进神经元存活、生长与突触可塑性的关键蛋白质。 |
| **Spatial Transcriptomics** | 空间转录组学 | Axis D | 在保持细胞空间位置信息的同时，测量其基因表达，实现“原位”解析。 |
| **scRNA-seq** | 单细胞 RNA 测序 | Axis D/B | 在单个细胞维度解析基因表达，揭示细胞群体的异质性 (Heterogeneity)。 |
| **PINNs** | 物理信息神经网络 | Axis F | 将生物物理规律（如扩散方程）嵌入神经网络，用于预测数据稀疏的生物过程。 |
| **pLMs** | 蛋白质语言模型 | Axis F | 利用 Transformer 学习蛋白质序列的“语法”，预测蛋白结构、功能与聚集倾向。 |
| **MNPs (Micro/Nanoplastics)** | 微塑料与纳米塑料 | Axis E | 环境中直径 <5mm (Micro) 及 <1μm (Nano) 的塑料碎片，能穿透 BBB 进入大脑。 |
| **Protein Corona** | 生物冠 | Axis E | 进入生物体的塑料颗粒表面吸附的一层血浆蛋白，决定了其生物学归宿与穿透力。 |
| **GSDMD (Gasdermin D)** | 细胞焦亡蛋白 | Axis B/E | 被激活后在细胞膜上形成孔道，引发细胞焦亡并导致 BBB 完整性破裂。 |
| **GENUS** | 感觉神经夹带激发 | Axis A/B | 通过 40Hz 的光/声刺激诱导大脑产生 Gamma 波，促进 Aβ 清除与免疫调节。 |
| **VIP (Vasoactive Intestinal Peptide)** | 血管活性肠肽 | Axis A/C | 40Hz 刺激下释放的关键神经肽，能扩张血管并增强动脉搏动，驱动类淋巴系统。 |
| **Ferroptosis** | 铁死亡 | Axis E/B | 一种依赖于铁的程序性细胞死亡形式，常由重金属暴露引发的氧化应激触发。 |
| **Histone Acetylation** | 组蛋白乙酰化 | Axis E/C | 一种表观遗传修饰，农药暴露可透过此机制长期“启动”免疫细胞。 |
| **Noise-induced Arousal** | 噪音诱导觉醒 | Axis E/A | 噪音导致蓝斑核释放去甲肾上腺素，物理性关闭类淋巴清除系统。 |
| **Digital Twin** | 数字孪生 | Axis D/F | 建立患者大脑的虚拟模型，整合多组学数据以模拟病程发展与药物反应。 |
| **AI Surrogate Brain** | 人工智能替代大脑 | Axis F | 利用数据驱动方法精确预测全脑动态行为，作为数字孪生的核心计算引擎。 |
| **Neural ODEs** | 神经常微分方程 | Axis F | 一种深度学习架构，用于建模随时间连续变化的生物系统（如蛋白聚集或脑网络演化）。 |
| **Epigenetic Mapping** | 表观遗传图谱 | Axis C | 记录基因表达受环境影响（而非基因序列改变）的修饰状态（如 DNA 甲基化）。 |
| **scMulti-omics** | 单细胞多组学 | Axis D | 同时在单个细胞中测量转录组、蛋白质组、表观组等多个维度的数据。 |
| **p-tau217** | 磷酸化 Tau 217 | Axis D/A | 血液中特定的 Tau 蛋白亚型，是目前最精确的失智症早期诊断生物标志物。 |
| **Bmal1 / Clock** | 昼夜节律基因 | Axis G | 调控生物钟的核心基因，影响微胶质细胞的吞噬能力与类淋巴系统的脉冲式清除。 |
| **Slow-wave Sleep** | 慢波睡眠 | Axis G | 深度睡眠阶段，是类淋巴清除系统运行的黄金窗口，对 p-tau 的清除至关重要。 |
| **Pericytes** | 周细胞 | Axis H | 包绕在毛细血管外的细胞，控制血流速度与 BBB 完整性，其丢失是血管性退行的起始。 |
| **Neurovascular Coupling** | 神经血管耦合 | Axis H | 神经元活动与局部血流量之间的精确匹配机制，AD 早期该机制会发生失灵。 |
| **Fibrinogen** | 纤维蛋白原 | Axis H/B | 血液中的凝血蛋白，一旦渗透进入大脑实质，会直接诱导微胶质细胞产生促炎反应。 |
| **Metabolic Reprogramming** | 代谢重编程 | Axis I | 神经元在应激下从正常的葡萄糖代谢转向脂肪酸氧化等损耗性路径的过程。 |
| **Insulin Resistance** | 胰岛素抵抗 | Axis I | 细胞对胰岛素信号不敏感，导致大脑“血糖饥饿”，触发 Tau 蛋白病理变化。 |
| **Neuronal Glycogen** | 神经元糖原 | Axis I | 正常情况下极少，在 AD 早期异常积累，被视为代谢紊乱与蛋白聚集的预警信号。 |
| **OCT-A** | 光学相干断层扫描血管造影 | Axis J | 非侵入性影像技术，可精确测量视网膜微血管密度，是大脑微循环状态的“窗口”。 |
| **RNFL** | 视网膜神经纤维层 | Axis J | 视网膜中的神经组织层，其变薄往往早于认知症状出现，是早期诊断的关键指标。 |
| **Cellular Senescence** | 细胞衰老 | Axis K | 细胞丧失分裂能力但仍维持代谢活性的状态，是慢性炎症与神经退行的底层驱动力。 |
| **SASP** | 衰老相关分泌表型 | Axis K | 衰老细胞分泌的促炎细胞因子、趋化因子与蛋白酶组合，会“感染”周围健康细胞。 |
| **Senolytics** | 衰老细胞清除药物 | Axis K | 一类能特异性诱导衰老细胞死亡的药物（如 D+Q），旨在重塑大脑微环境。 |
| **Epigenetic Clock** | 表观遗传时钟 | Axis K | 基于 DNA 甲基化水平预测生物年龄的方法，可用于评估大脑退行程度与干预效果。 |
| **LDAM** | 脂质积聚微胶质细胞 | Axis B/I | 含有大量脂滴的免疫细胞亚群，丧失了正常的吞噬功能，转而释放毒性脂质。 |
