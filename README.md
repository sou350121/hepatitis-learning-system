## ✍️ 研究者协作规范 (Researcher & Agent Protocol)

> **本手册采用 DocOps + AgentOps 混合范式。以下规则是维持知识库逻辑密度的核心，AI Agent 在执行任务时必须强制遵守。**

1. **拒绝平庸描述，直击生化底层**
   严禁记录模糊的定性表述。任何发现必须定位到具体的**分子路径、物理参数或计算模型**（例如：不再说“运动有益”，而说“Irisin 通过 PGC-1α 诱导 BDNF 重塑突触”）。

2. **多轴线标注与逻辑追溯 (The Logic Trace)**
   每一项新情报必须标明其所属的 `Axis` 及其对其他轴线的潜在影响。对于 NSC 旗舰论文，必须提取其核心的**逻辑/物理规律**（如：Tau 聚合的速率控制动力学）。

3. **动态链接与闭环，严禁信息孤岛**
   新内容进入时，必须自动检查并链接相关 Axis。例如：添加 Axis E（环境毒理）新发现时，必须寻找与 Axis H（血管屏障）和 Axis B（免疫）的潜在接口。

4. **逻辑可视化规范 (ASCII Visualization)**
   复杂的生物机制或系统交互必须配有简洁的 ASCII 逻辑流图。使用统一符号：`⚠️` (风险), `✨` (发现), `🚀` (转化), `🤖` (AI 介入)。

5. **结论先行，假设结尾 (Actionable Hypothesis)**
   每项情报记录必须以“核心结论”开篇，并以 1-2 个基于该发现的“可验证科学假设”结尾，确保知识库具有前瞻性的行动指引。
