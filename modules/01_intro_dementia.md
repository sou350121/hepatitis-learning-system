# 模組 01：腦退化病導論 (Module 01: Introduction to Dementia)

## 1. 定義與概覽 (Definition & Overview)
**腦退化病 (Dementia)**，臨床上常稱為**認知障礙症**，並非單一疾病，而是一組描述認知功能下降、嚴重程度足以干擾日常生活社交的症狀群。它主要影響記憶、思考、方向感、理解、計算、學習能力、語言和判斷力。

---

## 2. 常見類型 (Common Types)

### A. 阿茲海默症 (Alzheimer's Disease, AD)
- **佔比**：約 60-80% 的病例。
- **特徵**：最早期的症狀通常是難以記住最近發生的事件。
- **病理**：大腦中出現澱粉樣蛋白斑塊 (Amyloid-beta plaques) 和 Tau 蛋白纏結 (Tau tangles)。

### B. 血管性癡呆 (Vascular Dementia, VD)
- **病因**：由腦部血液供應受阻（如中風）引起的腦部損傷。
- **特徵**：認知下降往往呈現「階梯式」惡化。

### C. 路易氏體癡呆 (Dementia with Lewy Bodies, DLB)
- **特徵**：伴隨幻覺 (Hallucinations)、帕金森氏症狀（如僵硬、顫抖）以及注意力大幅波動。
- **病理**：神經元內出現 alpha-synuclein 蛋白質聚集。

---

## 3. 臨床分級量表 (Clinical Scaling)

### 全球退化量表 (Global Deterioration Scale, GDS)
| 等級 | 狀態 | 描述 |
| :--- | :--- | :--- |
| **GDS 1** | 無認知下降 | 正常健康狀態。 |
| **GDS 2** | 極輕微認知下降 | 偶爾忘記放鑰匙或人名（主觀感覺）。 |
| **GDS 3** | 輕微認知下降 | **輕度認知障礙 (MCI)**，在工作或社交場合被察覺。 |
| **GDS 4** | 中度認知下降 | **早期失智**，難以處理複雜財務或旅行計畫。 |
| **GDS 5** | 中重度認知下降 | **中期失智**，需要幫助選擇合適的衣物或記住地址。 |
| **GDS 6** | 重度認知下降 | 需要穿衣、如廁等日常生活輔助。 |
| **GDS 7** | 極重度認知下降 | 喪失語言與行走能力。 |

---

## 4. 研究方向 (Research Directions)
- **生物標記 (Biomarkers)**：如何透過血液檢測提前 20 年預測 AD？
- **神經保護 (Neuroprotection)**：延緩腦細胞凋亡的新型分子機制。
- **生活方式干預**：飲食 (MIND Diet) 與社交互動對認知儲備 (Cognitive Reserve) 的影響。
